关于飞传

快速、安全、跨平台的文件传输解决方案

项目简介

飞传 (FileFly) 是一款纯前端的跨设备文件互传工具，支持在局域网内快速传输文件，无需服务器中转，保护您的隐私安全。

项目采用现代前端技术栈，使用 WebRTC DataChannel 实现 P2P 直传，AES-GCM 256 位加密确保传输安全，文件分块技术支持大文件传输，实时进度显示让您随时了解传输状态。

使用方法

1. 打开应用

在文件管理器中找到 index.html 文件，直接双击打开它，或右键选择 "打开方式" → 选择浏览器。

2. 连接设备

应用加载完成后，点击 "设备连接" 页面，系统会自动扫描局域网内的设备。选择要连接的设备，点击 "连接" 按钮。

3. 传输文件

连接成功后，点击 "发送文件" 按钮，选择要传输的文件。系统会自动开始传输，并显示实时进度和速度。

4. 完成传输

传输完成后，系统会显示 "已完成" 状态。您可以在接收设备上查看传输的文件。

隐私政策

飞传非常重视您的隐私安全，我们采取了以下措施保护您的数据：

  • 所有文件传输均在局域网内直接进行，不经过任何服务器
  • 使用 AES-GCM 256 位端到端加密技术保护文件内容
  • 不会在任何设备上存储您的文件内容
  • 连接会话临时生成，传输完成后自动断开
  • 不收集任何个人信息或使用数据
