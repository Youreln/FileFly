快速开始

1

克隆项目

git clone https://github.com/Youreln/FileFly.git
2

安装依赖

cd FileFly && npm install
3

启动服务

npm start

启动后访问 http://localhost:3000 即可使用！

连接方式

局域网连接

确保设备连接同一WiFi，使用局域网IP访问

扫码连接

手机扫描二维码，一键跳转到传输页面

热点直连

电脑开启热点，手机连接后使用热点IP访问

USB共享

手机USB连接电脑，开启网络共享访问
